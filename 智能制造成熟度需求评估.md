### 产品功能架构梳理

#### 1. 数据集成与治理 (Data Integration & Governance)

| 二级功能模块 (Level 2 Module) | 三级功能模块 (Level 3 Module) | 核心功能说明 (Core Function Description) |
| :--- | :--- | :--- |
| **多源数据录入 (文件与手动录入)** | 信息化系统概况 | 支持通过 Excel/CSV 模板批量导入数据；对于无法自动获取的定性数据 (如战略文档)，提供手动填报入口。 |
| | 数据安全概况 | |
| | 流程适配性概况 | |
| | 其他 | |
| **数据治理与计算** | 指标计算引擎 | 内置上百个智能制造指标的计算逻辑 (如：设备联网率、OEE、数据自动采集率等)，自动将原始数据加工成评估指标。 |
| **自动化评估** | 成熟度等级判定 | 根据预设的模型和阈值，自动对每个评估项、维度和整体成熟度进行等级 (L1-L5) 判定。 |

#### 2. 智能评估与诊断模块 (Intelligent Assessment & Diagnosis Module)

| 二级功能模块 (Level 2 Module) | 三级功能模块 (Level 3 Module) | 核心功能说明 (Core Function Description) |
| :--- | :--- | :--- |
| **深度诊断分析** | 实时雷达图生成 | 自动生成并更新企业智能制造成熟度雷达图，直观展示各维度能力均衡情况。 |
| | 差距分析 | 自动对比企业当前状态与目标等级 (或行业平均水平) 的差距，并精准定位到具体的评估项。 |
| | 标杆对比 | 允许企业在匿名化后，与同行业、同规模的标杆企业数据进行横向对比，明确自身行业位置。 |
| | 专家意见 | 专家评估意见及报告调整。 |

#### 3. 报告生成 (Report Generation)

| 二级功能模块 (Level 2 Module) | 三级功能模块 (Level 3 Module) | 核心功能说明 (Core Function Description) |
| :--- | :--- | :--- |
| **报告生成** | 一键生成报告 | 用户可一键生成结构完整、图文并茂的《智能制造成熟度评估报告》(PDF/Word 格式)。 |
| | 报告模板自定义 | 允许管理员根据企业或项目需求，自定义报告的封面、Logo、章节结构和内容侧重。 |
| | 多版本对比报告 | 支持将不同时间点的评估报告进行对比，直观展示改进成效与能力演变。 |

#### 4. 系统管理 (System Management)

| 二级功能模块 (Level 2 Module) | 三级功能模块 (Level 3 Module) | 核心功能说明 (Core Function Description) |
| :--- | :--- | :--- |
| **用户与权限管理** | 用户管理 | 支持用户的增、删、改、查、禁用；支持批量导入。 |
| | 角色管理 | 支持基于角色的访问控制 (RBAC)，可自定义角色 (如：系统管理员、评估专家、企业管理员、普通查看者)。 |
| | 权限配制 | 可精细配置不同角色对菜单、功能、数据范围的访问和操作权限。 |
| **系统配制** | 评估模型管理 | 支持内置多种成熟度模型 (如 CMMI、IM3、《智能制造典型场景参考指引 (2025 年版)》)。并允许管理员对评估维度、等级和标准进行微调与启用/禁用。 |
| | 基础数据字典 | 维护行业类型、企业规模、指标单位等系统级基础数据。 |
| | 知识库管理 | 对“改进建议库”、“最佳实践案例库”等进行增删改查，为自动报告生成提供素材。 |
